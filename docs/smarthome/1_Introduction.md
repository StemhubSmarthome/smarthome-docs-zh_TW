# 簡介

## 簡介
SMARTHON micro:bit 智慧城市IoT學習套裝是專門為了教授物聯網而設的套裝。透過學習基本的電腦編程及電子知識，你可以利用物聯網，建設一個獨一無二的智慧城市。套裝內提供的Smarthon IoT:bit 開發板，能夠使用各種傳感器及驅動器，令你能夠在設計中實現多種有趣的功能。例如，使用傳感器去偵測交通狀況然後上載到互聯網，令你能夠作出統計或是控制等。
